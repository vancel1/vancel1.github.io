Emacs 24 的主题设置

By vancel1

发表于 Jun 24 2012

文章目录

如果你已经升级到 Emacs 24，那么你可以去掉 color-theme 了，现在已经内置了一个新的主题系统。

使用内建的主题

M-x customize-themes 选上你喜欢的主题，然后在 .emacs 里加上

自定义主题

如果你不喜欢自带的主题，你大可以以 /usr/share/emacs/24.x/etc/themes/ 目录下的主题为模板，然后修改成自己想要的样式。

主题文件命名方式为：*-theme.el ，你还需要自定义 custom-theme-load-path 函数，把主题文件放到你设定的目录下

ps: 你可以通过 “M-x list-faces-display” 查看当前主题的详细设置，又或者假如你想修改 org-mode 下的某个部分的颜色值，可以在这里面搜索一下需要改哪个值

pps: 你可以通过 “M-x load-file RET 你的主题文件地址” 即时查看主题的修改
