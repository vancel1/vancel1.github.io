地狱狂蛇2014 linux 下的设置

By vancel1

发表于 Aug 2 2015

文章目录

最近买了个灯厂的鼠标，linux 下插上就可以用了，问题是飘得有点蛋疼。windows 下这个鼠标的 dpi 是要到驱动里调的，而在 linux 下就连 razercfg 这个第三方驱动的支持列表里我也没有找到地狱狂蛇。

谷歌大法后，发现可以折中地在命令行改 xinput 设置，方法如下：

命令行输入： xinput list，找到你的鼠标的 device id

在我这里，我的鼠标的 device id 是 8，根据你的实际情况而定。然后输入： xinput list-props <你的 device id> 查看当前的设置：

找到 Device Accel Constant Deceleration (263) 这一项，记下括号里的数字（property id)。冒号后面的值越大表示速度越慢，例如我这里默认是 1.0 ，试了几次后我把它设置为 1.8 。命令行输入形如： xinput set-prop <你的 device id> <property id> <value>

想要保留设置的话，在 ~/.xinputrc (没有的话自己新建) 里加入上面那一行设置就可以了。
